# 我的博客系统

一个简单而功能完整的博客系统，使用纯前端技术构建。

## 功能特性

- ✅ 用户注册和登录
- ✅ 发布和管理博客文章
- ✅ 响应式设计
- ✅ 数据持久化存储

## 使用方法

1. 访问网站
2. 注册新账户
3. 登录系统
4. 点击"新建文章"开始写作

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage (数据存储)

## 部署说明

1. 将此仓库部署到GitHub Pages
2. 访问 `https://gbisaillonemarya6b-create.github.io/-my-blog-system/`

## 本地运行

直接打开 `index.html` 文件即可
